Personal Reflection

Interactive Prototypes Development

Through developing interactive media products, I've gained valuable experience in creating functional prototypes that can be tested with real users. The focus was on building interactive elements that respond to user input and provide meaningful feedback.

User Testing Process

The main challenge was conducting effective user tests on the interactive prototypes. I learned to observe user behavior, collect meaningful data, and iterate on the prototypes based on testing results. The Cardan project was particularly valuable for understanding how users interact with complex interfaces.

Skills Developed

Through this learning outcome, I've developed skills in:

  • Creating interactive prototypes with functional elements
  • Conducting structured user testing sessions
  • Iterating prototypes based on user feedback
  • Documenting the testing process and results
  • Integrating research insights into prototype development

Impact on Design Process

This experience has shown me the importance of testing interactive elements early and often. The ability to create testable prototypes and gather user feedback has become a core part of my design methodology for interactive media products.
